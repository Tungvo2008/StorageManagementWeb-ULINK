# Storage Management Web (Frontend)

React (Vite) UI tối giản để thao tác:

- Products
- Customers
- Sales (tạo đơn bán và trừ tồn kho)
- Invoices (xem HTML / tải PDF)

## Chạy dev

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

Mặc định UI gọi backend tại `VITE_API_BASE_URL=http://localhost:8000`.
